# Digit Recognition System Using ANN

A handwritten digit recognition web application built using TensorFlow, OpenCV, NumPy, and Streamlit.

## Features
- Upload handwritten digit images
- Image preprocessing using OpenCV
- Predict handwritten digits (0–9)
- ANN model trained on the MNIST dataset

## Technologies Used
- Python
- TensorFlow
- Streamlit
- OpenCV
- NumPy
- Pillow
